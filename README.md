# Vizuara Scientific ML Bootcamp - Teaching Assistant

AI-powered teaching assistant for the Vizuara Scientific Machine Learning Bootcamp.

## Setup

1. Clone the repository
2. Copy `.env.example` to `.env.local` and fill in your credentials
3. Run `npm install`
4. Run `npm run db:push` to set up the database
5. Run `npm run dev` to start the development server

## Deployment

Deploy to Railway with the environment variables from `.env.local`.
